# Invitación de Boda Digital

Este proyecto es una tarjeta de invitación digital interactiva creada con Python (Flask), HTML, CSS y JavaScript.

## Estructura
- `tarjeta.py`: Aplicación principal Flask.
- `templates/`: Archivos HTML.
- `static/`: Archivos CSS, imágenes y scripts.

## Ejecución Local
1. Instalar dependencias:
   ```bash
   pip install -r requirements.txt
   ```
2. Ejecutar la aplicación:
   ```bash
   python tarjeta.py
   ```
